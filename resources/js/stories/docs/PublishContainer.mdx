# PublishContainer
The `PublishContainer` component is used to create Publish Forms in the Control Panel. It handles field rendering, validation, localization and state management.

## Learn more
To learn more about the `PublishContainer` component (and publish forms in general), please visit the ["Publish Forms"](https://statamic.dev/control-panel/publish-forms) page on our main documentation site.
