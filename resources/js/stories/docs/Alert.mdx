# Alert
Alerts are used to display important messages, notifications, and feedback to users. They come in multiple variants to convey different types of information.

<Canvas of={AlertStories._DocsIntro} sourceState={'shown'} />

## Variants
Use the `variant` prop to change the appearance and semantic meaning of the alert. Available variants are `default`, `warning`, `error`, and `success`.

<Canvas of={AlertStories.Variants} sourceState={'shown'} />

## Custom Icons
You can override the default icon by passing a custom `icon` prop with the name of any available icon.

<Canvas of={AlertStories.CustomIcons} sourceState={'shown'} />

## Slot Content
Instead of using the `text` prop, you can use the default slot to provide custom content with HTML formatting.

<Canvas of={AlertStories.SlotContent} sourceState={'shown'} />

## Rich Content
Alerts support rich content including headings, paragraphs, and code snippets. This makes them perfect for displaying detailed information and instructions.
